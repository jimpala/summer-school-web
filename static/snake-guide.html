Snake

Overview

In this task we will be creating a simple 2D Snake game, with scope to add features such as multiplayer, a user interface, and power-ups.

Resources

Click here to access the Snake game guide.

Atom Instructions

We will be coding this game using the Atom text editor.

Create a folder on the desktop to contain all of your project files in, and open it in Atom.

In order to test your code, you will need to run a server in the browser by using the command "CTRL+SHIFT+3". Once you have done this once, you don't need to do it again- just refresh the web page to test your new code. Be sure to save your code before testing so that the web page updates correctly.
